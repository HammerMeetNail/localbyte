# ByteBites Recipe Book (Demo)

ByteBites is a fictional, client-side recipe library demo by LocalByte LLC. It showcases fast search, multi-facet filtering, favorites, print-friendly recipe pages, ingredient scaling, and a step-by-step cooking mode.

**Privacy:** This demo stores favorites and filter preferences in your browser via `localStorage`. Nothing is sent to a server.

## Run locally

From the repo root:

- `make local`
- Visit `http://localhost:9000/demos/bytebites/index.html`

## Tests

- Unit: `npm run test:unit`
- E2E: `npm run test:bytebites`

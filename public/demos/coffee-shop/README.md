# Cedar & Steam Coffee (Demo)

Fictional coffee shop website demo built by LocalByte LLC to showcase design + development quality.

## Local development

From the repo root:

```bash
make local
```

Then open:

- `http://localhost:9000/demos/coffee-shop/index.html`

## Running tests (optional)

This demo includes Playwright end-to-end tests.

```bash
npm install
npm test
```

## Notes

- This is a **fictional** business and the content is for demonstration only.
- All visual assets in `public/demos/coffee-shop/assets/img/` are original demo assets created for this repo.
